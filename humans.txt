// MISSION
LiveScoreFree is a free, community-powered sports scoreboard providing real-time
live scores, match commentary, player lineups, teams statistics, and results for
football, cricket, basketball, tennis, and all major sports leagues worldwide.

// TECHNOLOGY
Built with: Vanilla JavaScript, Service Workers, Progressive Web App
Hosted on: Cloudflare Global Network
Data Sources: ESPN API, TheSportsDB
Frontend Framework: None (Lightweight vanilla JS)
Performance: 100% client-side rendering, offline support

// PRINCIPLES
- Free and accessible to all users worldwide
- No paywall, no ads, community-supported
- Fast, lightweight, and mobile-optimized
- Privacy-focused: no tracking, no data collection
- Open to community contributions

// FEATURES
✓ Live Scores (Real-time updates)
✓ Trending Matches (Community engagement)
✓ Upcoming Matches (Schedule)
✓ Match History (Results archive)
✓ Top Leagues (Featured competitions)
✓ Multi-Sport Support (Football, Cricket, Basketball, Tennis, etc.)
✓ Mobile Responsive (PWA ready)
✓ Offline Support (Service Worker)
✓ Dark/Light Themes
✓ Multi-Language Support
